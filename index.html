1. Descarga e Instalación

Para Celular:

  1. Descarga el archivo APK desde el botón "Descargar para Celular"
  2. Ve a Configuración > Seguridad > Permitir instalación de fuentes desconocidas
  3. Instala la aplicación descargada
  4. Abre FASTNET TV

Para Android TV:

  1. Descarga el archivo APK desde el botón "Descargar para Android TV"
  2. Transfiere el archivo a tu Android TV via USB o descarga directa
  3. Instala usando un explorador de archivos
  4. Abre FASTNET TV desde el menú de aplicaciones

2. Configuración Inicial

Al abrir la aplicación por primera vez:

  • Ingresa tu usuario y contraseña proporcionados
  • La aplicación se configurará automáticamente
  • Espera a que cargue la lista de canales
  • ¡Listo para disfrutar!

3. Navegación y Uso

Canales en Vivo:

  • Navega por categorías: Deportes, Noticias, Entretenimiento, etc.
  • Usa el control remoto o toca la pantalla para cambiar canales
  • Presiona OK/Enter para ver información del programa

Películas y Series:

  • Accede desde el menú principal
  • Busca por género, año o título
  • Selecciona y disfruta en alta calidad

4. Funciones Avanzadas

  • Favoritos: Marca tus canales preferidos
  • EPG: Guía de programación de 7 días
  • Grabación: Graba tus programas favoritos
  • Control Parental: Bloquea contenido para niños
  • Multi-idioma: Cambia audio y subtítulos

5. Solución de Problemas

Si la aplicación no carga:

  • Verifica tu conexión a internet
  • Reinicia la aplicación
  • Limpia caché de la aplicación
  • Contacta soporte técnico

Si hay cortes o buffering:

  • Verifica velocidad de internet (mínimo 5 Mbps)
  • Cambia calidad de video en configuración
  • Reinicia tu router
  • Usa conexión por cable ethernet si es posible
